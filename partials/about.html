About

This project was created in the University of Tartu during the MTAT.03.238 Advanced Algorithmics fall 2015 course in the Institute of Computer Science by @TonisPool. I wanted to create a web visualization of the Ant Colony Optimisation algorithm. At the time, there were a couple of ACO visualisations out there, but none that I could find that ran in the browser, which makes running the visualisation much easier. Hopefully this helps people in understanding the algorithm and exploration of what happens with different variable configurations. You can read more about ACO from the algorithm page.

The code is hosted on GitHub - https://github.com/poolik/visual-aco - and the poster that gives an overview of the project is here.

The project uses the two.js for drawing the elements and AngularJS with Bootstrap for the rest of the webapp. For implementing the actual Ant System algorithm some ideas were borrowed from here and here.
